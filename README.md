# Ecommercea Application

This is a basic mobile E-commerce project for educational purposes and released on Google Play Store

### Technologies

- Mvvm
- Navigation 
- Flow
- Dagger - Hilt
- Coroutines
- Firebase Auth & Firestore
- Glide
- ViewPager2
